## Run Locally

Clone the project

```bash
  git clone https://github.com/feliperocha93/challenge-infleux-fe
```

Go to the project directory

```bash
  cd challenge-infleux-fe
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
